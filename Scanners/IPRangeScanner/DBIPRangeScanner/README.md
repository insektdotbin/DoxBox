# DoxBox IP Range Scanner
## Overview
The IP Range Scanner scans IP ranges and ports using multi-threading. It supports CIDR notation and start-end IP ranges.    

## Features
+ Scans IP ranges and ports.
+ Multi-threaded for fast scanning.
+ Detects open, closed, and filtered ports.

## Installation
1. Install Python 3.8+.
2. Install dependencies:

```bash
pip install art colorama
```
3. Download the Project:

```bash
git clone https://github.com/insektdotbin/DoxBox
```
# Usage
```bash
python dbipscanner.py
```
+ Enter an IP range (CIDR or start-end format).
+ Specify ports to scan.

## Example:
```
Choose input method (1/2): 1
Enter CIDR notation: 192.168.1.0/24
Enter ports (e.g., 80,443 or 1-100): 80,443
```
