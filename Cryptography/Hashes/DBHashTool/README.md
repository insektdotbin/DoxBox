# DoxBox Hash Tool
## Overview
The Hash Tool converts text into various hash formats. It supports MD5, SHA-1, SHA-2, SHA-3, and BLAKE2 algorithms. The tool warns against using insecure hashes like MD5 and SHA-1.

## Features
+ Supports 9 hash algorithms.
+ Warns against insecure algorithms.
+ Easy-to-use console interface.

## Installation
1. Install Python 3.8+.
2. Install dependencies:

```bash
pip install art colorama
```
3. Download the Project:

```bash
git clone https://github.com/insektdotbin/DoxBox
```
# Usage
```bash
python dbhashtool.py
```
1. Select a hash algorithm (e.g., SHA-256).
2. Enter the text to hash.

## Example:

```
Choose algorithm (1-9, 0 to exit): 3
Enter text to hash: Hello World!
```

# Ethical Guidelines
Use this tool responsibly.
Avoid using insecure hashes for sensitive data.
